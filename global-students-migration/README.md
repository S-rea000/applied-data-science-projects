# Global Students Migration Analysis

This project analyzes global student migration patterns using Python and pandas.

## Files Included
- `global-students-migration.ipynb` — Data analysis notebook
- `global_students_migration.csv` — Dataset used for the analysis

## Objectives
- Explore student migration trends
- Understand country-wise movement patterns
- Perform basic data exploration and insights

## Tools Used
- Python
- Pandas
- Google Colab

## Skills Demonstrated
- Data loading and preprocessing
- Exploratory data analysis
- Working with real-world datasets
