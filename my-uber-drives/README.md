# Uber Drives Data Analysis

This project explores and analyzes Uber trip data using Python and pandas. It focuses on understanding travel patterns, trip purposes, and distance trends through basic exploratory data analysis.

## Files Included
- `my-uber-drives.ipynb` — Main analysis notebook
- `uber_drives.csv` — Dataset used for the project

## Objectives
- Clean and preprocess trip data
- Analyze trip categories and purposes
- Study distance distribution
- Explore time-based travel patterns

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Google Colab / Jupyter Notebook

## Skills Demonstrated
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Data visualization
- Working with real-world datasets

## Dataset Features
The dataset includes:
- Start Date
- End Date
- Category
- Start Location
- Stop Location
- Distance
- Purpose

## Summary
This project provides insights into how Uber rides vary based on purpose, distance, and time. It serves as a foundational data analysis project for learning real-world dataset handling and visualization.
